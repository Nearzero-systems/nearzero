# Nearzero Security Policy

At Nearzero, security is a top priority. We appreciate the help of security researchers and the community in identifying and reporting vulnerabilities.

## Reporting a Vulnerability

If you have discovered a security vulnerability in Nearzero, we ask that you report it responsibly by following these guidelines:

1. **Contact us:** Send an email to [contact@nearzero.dev](mailto:contact@nearzero.dev).
2. **Provide details:** Include a clear description of the vulnerability, steps to reproduce, and potential impact.
3. **Allow time to fix:** Give us reasonable time to investigate and address the issue before public disclosure.

## Scope

This policy applies to the Nearzero platform and related services hosted at [nearzero.dev](https://nearzero.dev).

Thank you for helping us keep Nearzero secure for everyone.
