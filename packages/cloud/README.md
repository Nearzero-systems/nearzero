# @nearzero/cloud

Private Nearzero Cloud package. This package is **not** part of the Community
open-source release published to [Nearzero-systems/nearzero](https://github.com/Nearzero-systems/nearzero).

Hosted Nearzero installs this package from a private repository and wires it through
`bootstrapCloudEdition()` plus the cloud platform router extensions.

Community operators should never install or import this package.
